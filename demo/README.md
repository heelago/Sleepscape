# Browser Demo

`sleepscape.html` is a standalone HTML/Canvas/Web Audio prototype of Sleepscape. Open it in any modern browser to try the core drawing and ambient sound experience — no Xcode or iPad required.

This was the original proof-of-concept that preceded the native Swift/Metal app. It includes touch drawing with glow effects, symmetry modes, ripple particles, a breathing guide, and generative ambient audio.

**To try it:** open `sleepscape.html` in your browser, or visit it via GitHub Pages if enabled.

> Note: The native iPad app (the rest of this repo) is the full version — Metal shaders, AudioKit audio, 120fps rendering. This demo is a simpler approximation.
